# User Stories Summary Task

Analyze user story files and create a concise summary.

## Task

Read all user story files and generate a brief overview of what stories were created.

## Required Output Format

```markdown
## User Stories Summary

**Total Stories**: [number]

**Stories Created**:
- **[Story Title]**: [One sentence describing the story's value/goal]
- **[Story Title]**: [One sentence describing the story's value/goal]
- **[Story Title]**: [One sentence describing the story's value/goal]
[... for all stories]

**Coverage**: [1-2 sentences describing overall scope and focus areas covered]
```

**OUTPUT ONLY THE MARKDOWN ABOVE - NO OTHER TEXT**

---

## Input Data

**User Story Files Directory**: [[PLAN_PATH]]user-stories/

Please read all .md files in that directory and summarize them.
