# Command: /init-specs

Initialize project specifications with sensible defaults based on Black Box Architecture and pragmatic TDD.

## Usage
`Run the command defined in .planning/commands/init-specs.md --language <lang> --framework <frame>`

## Full Prompt
```markdown
# Initialize Project Specifications

Generate standardized development specification files for a project with sensible defaults based on Black Box Architecture and pragmatic TDD principles.

## Arguments
- `--language <language>`: Programming language (e.g., typescript, python, go, rust, java)
- `--framework <framework>`: Framework (e.g., nextjs, react, django, rails, spring)
- `--update`: Re-run project discovery config detection only (skip spec file generation)
- Additional context: Any other text will be used as additional context for customization
```
